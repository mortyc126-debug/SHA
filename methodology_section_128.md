## КЛЮЧЕВЫЕ ТЕОРЕМЫ

### T_CARRY_PATTERN_DUAL (П-782) ★★★★★

**Утверждение:** SHA-256 имеет две разные carry-signatures в последних словах выхода.

```
H[6][b31..b28]: phi = (−0.21, −0.12, +0.22, −0.12) → паттерн --+-
H[7][b31..b28]: phi = (−0.03, +0.14, +0.14, −0.05) → паттерн -++-
H[5][b31..b28]: phi ≈ 0                              → паттерн 0000
```

**Механизм:**
```
H[7] = h[63] + IV[7]    h[63] = g[62] (прямо)
H[6] = g[63] + IV[6]    g[63] = f[62] = e[61] (через 2 раунда)
H[5] = f[63] + IV[5]    f[63] = e[62] (через 3 раунда)
```

carry[63]=0 → h[63] смещён → H[7] получает паттерн.
h[63] через раунд 63 влияет на e[63]... → H[6] получает другой паттерн.
H[5] = f[63] = e[62] — слишком далеко, сигнал растворяется.
