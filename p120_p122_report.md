## НОВЫЕ ТЕОРЕМЫ

**T_ZERO_H_CHAIN** (П-120, аналитически + верифицирована):
```
δh[18] = δh[19] = δh[20] = 0  детерминированно в Wang-паре.
Цепочка: δh[r] = δg[r-1] = δf[r-2] = δe[r-3].
δe[17..14] = 0 из Wang → δh[18..20] = 0.
δh[21] = δe[18] ≠ 0 — первый ненулевой h.
```

**T_TRIANGULAR_SYSTEM** (П-120):
```
Система δe[18..k]=0 треугольна:
  δe[18] = D + W18     → D = -W18
  δe[19] = C + f(D...) → C = ...
  δe[20] = B + f(C...) → B = ...
  δe[21] = A + f(B...) → A = ...
Каждое уравнение добавляет ровно одну новую переменную.
НО: {A,B,C,D} = {δa[17..14]} — СОСТОЯНИЕ, не вход.
Нельзя задать произвольно.
```

**T_W12_NEUTRAL_DE17** (П-122, P=1.0/10000):
```
P(de[17] preserved | varying Wn[12]=Wf[12]) = 1.0000
Wn[12] нейтрален для de[17] если оба меняются одинаково.
```

**T_W12_AFFECTS_DE18** (П-122, HW~16):
```
E[HW(de18 XOR de18')] при вариации Wn[12] = 16.04/32
Wn[12] влияет на de[18] свободно.
→ Нейтральные биты de[17] дают независимые значения de[18].
```

**T_SOL_CASCADE** (П-122, теоретически + экспериментально):
```
|Sol_17| >= 2^96  [P-93, доказано]
|Sol_18| >= 2^64  [фильтрация из Sol_17 × 2^-32]
|Sol_19| >= 2^32  [фильтрация из Sol_18 × 2^-32]
|Sol_20| >= 1     [фильтрация из Sol_19 × 2^-32]

ПРОБЛЕМА: фильтрация стоит O(2^96) total, что хуже O(2^32)!
Каскад не ускоряет поиск коллизии.
```

## СТРАТЕГИЧЕСКИЙ ИТОГ П-116..П-122

```
ЧТО МЫ ДОКАЗАЛИ:
  - Wang-цепочка: δe[2..16]=0 за O(1)  ← детерминированно
  - Барьер r=17: P(δe[17]=0) = 2^-32   ← birthday O(2^32)
  - δe[r] для r>=18: независимы, P=2^-32 каждый
  - T_TRIANGULAR_SYSTEM: красивая структура, но не решаема
    через свободные параметры
  - T_SOL_CASCADE: нейтральные биты расширяют, но не ускоряют

ФУНДАМЕНТАЛЬНЫЙ БАРЬЕР:
  Для полной коллизии нужны зависимости между δe[r] и δe[r+1].
  P(оба=0) >> 2^-64 — вот что нужно найти.
  
  Это задача дифференциального трейл-поиска.
  Li et al. 2022 достигли 31 раунда именно этим методом.

НАШ ВКЛАД В НАПРАВЛЕНИЕ АТАКИ:
  T_DEAD_ROUNDS объясняет почему раунды 0..16 бесплатны.
  Для раундов 18..64: нужна теория зависимостей δe[r].
  Это открытая задача дифференциальной криптографии.
```
