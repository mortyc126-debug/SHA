## НОВЫЕ ТЕОРЕМЫ

**T_STATE17_FGH_ZERO** (П-119, верифицирована 100 000 пар):
```
В любой Wang-паре:  δf[17] = δg[17] = δh[17] = 0  детерминированно.
Причина: f_new=e_old → δf[17]=δe[16]=0 (Wang обнулил δe[14..16]).
Эффективная размерность δstate[17]: 160 бит (не 256).
При δe[17]=0: 128 бит.
```

**T_DE18_FORMULA** (П-119, аналитически):
```
δe[18] = δa[14] + δW64[18]  при δe[17]=0, δh[18]=0.
Упрощение: δh[18]=δg[17]=0 из T_STATE17_FGH_ZERO.
```

**T_DE18_INDEPENDENCE** (П-119, измерено):
```
corr(δa[14], δW64[18]) = 0.006 ≈ 0.
→ P(δe[18]=0) = 2^-32 независимо.
```

**T_W64_DEAD_EXTENDED** (П-118):
```
При δW[0]=1 XOR: δW64[17]=δW64[19]=δW64[21]=0 аналитически.
Структура расписания создаёт "мёртвые" W64-раунды.
НО: δe распространяется через S1(e) независимо от δW64.
```
