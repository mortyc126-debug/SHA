## 0.2 Объекты, которых нет в существующей математике

### 0.2.1 Новые алгебраические структуры

**1. Двойное кольцо (Bi-Ring) R₃₂ = (Z/2³², ⊕, +)**

Множество 32-битных слов с ДВУМЯ операциями сложения:
- XOR (⊕) — сложение в GF(2)³²
- ADD (+) — сложение в Z/2³²

Это НЕ кольцо (нет единого дистрибутивного закона). Это НЕ near-ring.
Ключевое свойство: **невозможность одновременной линеаризации** (F39).
Не изучалось как алгебраическая структура.

**2. TLC-декомпозиция ΔH = L ⊕ Q ⊕ C**

Каноническое разложение дифференциала хеша на три слоя:
- L (линейный): XOR-компоненты, rank = 256
- Q (квадратичный): Ch/Maj вклады
- C (carry): нелинейные переносы

Три слоя **статистически независимы** (F11) и дают точное разложение (F1).
Аналогов нет: не гомологическая фильтрация, не градуировка по степени.

**3. p-адическая башня Sol_k**

Вложенная последовательность множеств решений:
```
Sol_1 ⊃ π(Sol_2) ⊃ π(Sol_3) ⊃ ... ⊃ π(Sol_32) ∋ collision
```
где Sol_k = {δW : F(W, W+δW) ≡ 0 (mod 2^k)}.

Ключевое открытие: **башня несвязна** (T_TOWER_INDEPENDENCE) —
Sol_{k+1} НЕ является подъёмом Sol_k. Hensel lifting неприменим.
Это принципиально новый тип p-адической обструкции.

### 0.2.2 Новые функции и метрики

**4. Профиль случайности R(H)**

Вектор-функция хеш-функции H:
```
R(H) = (DEP, τ, Λ, Ω)
```
- DEP[r] = размерность детерминированного пространства на раунде r
- τ = номер раунда «пробуждения» (transition round)
- Λ = ширина слова (= w, универсальный закон T_LAMBDA_EQUALS_W)
- Ω[r] = стоимость обнуления δe[r]

Не существует в литературе. Ближайшее — diffusion profile, но R(H) включает алгебраическую информацию.

**5. Barrier function b(j)**

```
b(j) = min{r : P(δe[r] ≠ 0 | δW[j] ≠ 0) > 0}
```
Точный результат: b(j) = j + 1 (T_BARRIER_SHIFT).
Свойства: b(j₁ + j₂) = min(b(j₁), b(j₂)) (T_INTERFERENCE_MIN).

Это **тропическая полу-метрика** на пространстве дифференциалов.
Связь с тропической геометрией не исследована.

**6. Серая зона (Gray Zone)**

Переходная область из 2 раундов после барьера:
```
P(δcarry = 0 | r) = [1.0, 1.0, ..., 1.0, 0.593, 0.527, 0.500, 0.500, ...]
                      ←— детерм. —→  ←gray→  ←— случайный —→
```
Универсальная длина = 2 (T_GRAY_LENGTH). Константы 0.593, 0.527 — новые.
Аналогов нет: не фазовый переход (дискретен), не перколяция.

**7. Carry determinism coefficient D(r)**

```
D(r) = P(δcarry[r] = 0 | δstate[r] = 0)
```
Профиль: [1.0^15, 0.593, 0.527, 0.500^47].
Это новая мера «алгебраической чистоты» раунда.

### 0.2.3 Новые конструкции

**8. Адаптивный каскад (Adaptive Cascade)**

Алгоритм: для каждого r = 3..16, выбрать W[r-1] так чтобы δe[r] = 0.
```
W_adapted[r-1] = W_natural[r-1] − δe_before[r] (mod 2^32)
```
Sensitivity = 1 (T_SENS_ODD_UNIVERSAL): Newton сходится за 1 шаг.
Это **конструктивный обратный оператор** для нелинейной системы.
Аналогов нет: не метод Ньютона (конечная сходимость), не lifting.

**9. Dead Rounds**

Раунды r < first_use(W[k]) где DP(state, δW[k]) = 1.0 тождественно.
Дифференциал «спит» — **детерминистическая пропагация** без потерь.
Это структурное свойство Merkle-Damgård, не описанное в литературе.

**10. Цикловая аномалия проекции**

g_n : {0,1}^n → {0,1}^n — проекция SHA-256 на n бит.
При n=32: 4 цикла вместо ожидаемых 22 (T_CYCLE_ANOMALY).
LCM длин << теоретический (2^{28.3} vs 2^{31.6}).

Это **проекционное вырождение** (T_PROJECTION_DEGENERATION):
проекция биекции f: {0,1}^256 → {0,1}^256 на k бит
систематически уменьшает число циклов.
