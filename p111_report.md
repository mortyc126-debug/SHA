## EXP B: Wang δ vs случайный δ — DEP профили

### MD5: δM[4] = 2^31 (Wang-стиль)

```
r    DEP(rand)   DEP(Wang)   отношение
1      2.45%       0.00%      0.000
2      8.35%       0.00%      0.000
3     20.49%       0.00%      0.000
4     41.36%       0.00%      0.000
5     63.02%       2.47%      0.039   ← Wang в 25× медленнее!
6     82.40%       8.90%      0.108
7     95.14%      21.25%      0.223
8     99.08%      41.92%      0.423
9     99.93%      63.59%      0.636
10    99.99%      81.66%      0.817   ← Wang r*

MD5 Wang-G5 = 64/10 = 6.4
```

### SHA-256: δW[0] = 0x8000 (наш Wang)

```
r    DEP(rand)   DEP(Wang)   отношение
1      2.10%       2.47%      1.178   ← Wang чуть быстрее!
5     88.15%      89.67%      1.017   ← Wang r*

SHA-256 Wang-G5 = 64/5 = 12.8  (= G5(rand))
```

### Вывод сравнения

```
MD5:    Wang δ диффундирует в ~25× медленнее случайного
SHA-256: Wang δ ≈ случайный (разница <2%)

SHA-256 S1(e) = ROTR(6,11,25) разрушает любую структуру δe
за один шаг — независимо от формы дифференциала.

MD5 F(b,c,d) = (b&c)|(~b&d) сохраняет структуру δ
при специально подобранных условиях на биты.
```

## EXP C: G8 — настоящий предиктор

### Почему Wang-G5 ≈ G5(rand)?

```
Wang-G5(MD5)    = 6.4  vs  G5(rand) = 9.1
Wang-G5(SHA-256) = 12.8 vs  G5(rand) = 12.8

Wang δ медленнее чем rand для MD5 (6.4 < 9.1) — есть разница!
Но разница НЕ разделяет взломанные и устойчивые хеши.
```

### Гипотеза G8

```
G8: Уязвимость к Wang-атакам определяется
P(δf(x,y,z) = 0 | δx = const)
где f — функция раунда.

Если P >> 1/2^w → функция "прозрачна" для дифференциалов.

MD5 F(b,c,d) = (b&c)|(~b&d):
  При δb=0: δF = δc при b=1, δF = δd при b=0
  → P(δF=0 | δb=0, δc=0) = 1   ← полная прозрачность!
  → Wang строит sufficient conditions через δb=0 "бесплатно"

SHA-256 Ch(e,f,g) = (e&f)|(~e&g):
  Та же структура, НО перед Ch стоит S1(e):
  S1(e) = ROTR(e,6)⊕ROTR(e,11)⊕ROTR(e,25)
  Для любого δe: δS1(e) псевдослучаен → δT1 случаен
  → P(δCh≈0 | δe≠0) ≈ 2^-32  ← нет прозрачности
```

### Формализация G8

```
Определим DP(f, δx) = P(δf(x,y,z)=0 | δx=const, random y,z)

MD5 F:    DP_max = max_{δx} P(δF=0|δx) = 1.0  (взломан)
SHA-1 F0: DP_max = 1.0  (взломан)
SHA-256 Ch+S1: DP_max ≈ 2^-32  (устойчив)

G8: если DP_max(f_round) > 1/2^(w/2) → хеш уязвим к Wang
```
