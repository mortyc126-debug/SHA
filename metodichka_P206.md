# П-206: Три пути к ΔW[17] — итоговые результаты

## Контекст
П-205 показал: MILP в П-204 нашёл тривиальное решение (ΔW[0]=0).
Реальный результат: min HW(ΔW[17]) = 3 (числово, 700K пар).
П-206 — три параллельных пути для уточнения картины.

---

## Путь 1: Исправленный MILP

**Версия 1** (constraint: ΔW[0]≥1):
```
Статус: Optimal, 0.1 сек
HW(ΔW[0])=1, HW(ΔW[1])=0, HW(ΔW[2])=0 → HW(ΔW[17])=0
```
**Снова тривиально**: при ΔW[1]=ΔW[2]=0, ΔW[17]=0 тождественно.

**Версия 2** (constraint: ΔW[0]≥1, ΔW[1]≥1, ΔW[2]≥1):
```
Статус: Optimal, 4.0 сек
HW(ΔW[0])=1, HW(ΔW[1])=14, HW(ΔW[2])=10 → HW(ΔW[17])=0
```
Формально ноль, но (ΔW[1], ΔW[2]) — свободные переменные, не Wang chain.

---

## Путь 2: Структурный анализ

**Новый теоретический результат (верифицирован):**

При δW[1]=δW[2]=0 (аддитивно):
```
ΔW[17] = sig1(ΔW[15]) + ΔW[10] + sig0(ΔW[2]) + ΔW[1]
       = sig1(0) + 0 + sig0(0) + 0 = 0
```
**Доказательство**: sig0(0) = sig1(0) = 0 (ROTR и SHR от нуля = ноль).

**Следствие**: барьер ΔW[17] ≠ 0 существует ТОЛЬКО из-за того, что Wang chain
производит ненулевые ΔW[1] и ΔW[2] через нелинейность S1 в раундах 1,2.

---

## Путь 3: Прямой перебор по формуле

**Результаты** (формула ΔW[17] = sig0(ΔW[2]) + ΔW[1]):

| Ограничение | HW(ΔW[17]) | Метод |
|-------------|------------|-------|
| ΔW[2]≠0, ΔW[1]=−sig0(ΔW[2]) | 0 | Аналитически (всегда) |
| ΔW[1]≈S1(ΔW[0]) (Wang approx) | **2** | 5M×5 пар, 53с |
| Систематический: HW(dw1)∈{1..5} | **1** | 1.7M комбинаций |

**Лучший реальный Wang-приближённый результат** (HW=2):
```
dw0 = 0xfe1691c6
dw1 = 0xd7908416  (≈ -S1(dw0))
dw2 = 0x0c348fc5
sig0(dw2) + dw1 = 0x80004000  HW=2 ✓
```

---

## Новые математические факты (П-206)

1. **δW[1]=δW[2]=0 ⟹ ΔW[17]=0** — тождественно (доказано аналитически + верифицировано 100%)

2. **Настоящий барьер**: Wang chain вынужден производить ΔW[1]≠0, ΔW[2]≠0
   из-за нелинейности S1. Именно это поднимает HW(ΔW[17]) с 0 до 3.

3. **Нижняя граница**: при S1-приближении Wang — HW(ΔW[17])=2 достижимо.
   Реальный Wang chain (с δe=0) может быть чуть хуже или лучше.

4. **MILP правильная постановка** = 16-раундовый, нелинейные carry,
   ~6000 переменных, академическая лицензия Gurobi.

## Задача П-207

Измерить распределение (ΔW[1], ΔW[2]) для реальных Wang chain пар:
- Взять 10K пар из П-97 (Wang MT поиск)
- Построить гистограмму HW(ΔW[1]) и HW(ΔW[2])
- Для пар с малым HW(ΔW[1]) — искать ΔW[0] минимизирующий ΔW[17]
- Цель: подтвердить или опровергнуть HW(ΔW[17])=2 реально

**Это числовая задача, не требует Gurobi.**
