## Блок 5: Фундаментальное уравнение

F25. C = 2^{n/2} × (1 + η^{n_eff}/n)
       = 2^128 × (1 + 10^{-10000})

Все формулы выводятся из ОДНОЙ константы:
  η = 1 - H(1/4) = 1 - 0.811 = 0.189
  "Постоянная тонкой структуры" carry-криптографии

## Связи

η → H_⊥ = n(1-η)
η → ξ = 1/ln(1/(1-2η))
η → d(⊕,+) = 1 - (1-η)^{2n}... нет: d = 1-(3/4)^n, η связана через H(1/4)

Точная связь:
  P(carry=1 at position 0) = 1/4
  H(1/4) = 0.811 = 1 - η
  I(c_i; c_{i+1}) = η = 0.189
  ξ = 1/ln(2) (из gap = 1/2 марковской цепи)
  k* = log₂(w) (из branching ротации)
