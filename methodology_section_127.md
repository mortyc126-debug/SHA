## КЛЮЧЕВЫЕ ТЕОРЕМЫ

### T_H6_B31 (П-751) ★★★★★

**Утверждение:** H[6][b31]=0 при carry[63]=0 имеет phi=+0.215.

**Данные (N=2000, K=64):**
- P(H6[b31]=0 | carry=0, K64) = **0.714**
- P(H6[b31]=0 | random) = 0.499
- phi = **+0.215**, Z = **19.3**

**Контекст:** H[6] = f[62] + IV[6] = e[61] + IV[6]. Bit 31 — старший бит. При carry=0 финальное сложение e+d дает результат с определённым старшим битом.

---

### T_H6_PATTERN_4 (П-752) ★★★★★

**Утверждение:** Паттерн 4 бит H[6] создаёт новый мощный distinguisher.

**Предикат:** H[6][b31]=0 ∧ H[6][b30]=0 ∧ H[6][b29]=1 ∧ H[6][b28]=0

**Данные (N=1000, K=64):**
```
P(паттерн | SHA-256, K64) = 0.263
P(паттерн | random)       = 0.0625 (= 1/16)
Advantage                 = +0.200
Z                         = 26.2
```

**Механизм:** Паттерн (0,0,1,0) в битах 31-28 — carry-signature. При carry[63]=0 финальное сложение (h[63]+IV[7]) создаёт предсказуемый carry-паттерн в H[6] через f[62]→e[61] цепочку.

---

### T_CHAIN_Q93 (П-755) ★★★★

**Утверждение:** Управляемая двухблочная цепочка частично сохраняет phi.

**Схема:**
```
Стандартный IV → HC K=64 → carry=0 → H1
H1 как новый IV → HC K=64 → carry=0 → H2
```

**Данные (N=300 пар блоков):**
| Величина | Блок 1 | Блок 2 |
|----------|--------|--------|
| P(H[6][b29]=1) | 0.723 | **0.557** |
| phi(H[6][b29]) | +0.223 | **+0.057** |
| phi(H[7][b29]) | +0.130 | +0.037 |

**Следствие:** phi затухает в ~4× за один блок, но не обнуляется. Это означает что управляемая цепочка из n блоков имеет phi ≈ 0.22 × (1/4)^(n-1). При n=3: phi ≈ 0.013 — неизмеримо.

**Вывод:** Q93 частично решён. Цепочка работает на 1 переходе.
