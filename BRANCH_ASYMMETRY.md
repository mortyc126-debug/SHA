# Структурная асимметрия e/a-ветвей SHA-256

## Теорема: Самореференция a-ветви

e_new = d + T1 — ДВА независимых источника (d из a-сдвига, T1 из раунда)
a_new = T1 + T2 — T2 = Σ₀(a) + Maj(a,b,c) зависит от a → САМОРЕФЕРЕНЦИЯ

При δa = 0: δT2 = 0 автоматически. δa_new = δT1 = δe_new.
Нельзя обнулить δa, не обнулив δe одновременно.

## Инвариант φ = δa - δe

φ[r+1] = Σ₀(δa[r]) + δMaj[r] - δa[r-3]

НЕ зависит от δW[r] (T1 сокращается). Жёсткое условие.
φ-chain невозможен: нет свободного параметра для управления φ.

## Рекурсия δa

δa[r+1] = -δa[r-3] + Σ₀(δa[r]) + Maj_r
Автономная 4-го порядка. Начальное условие: δa[1] = δW₀ (32 бита).
Все δa[1..16] — детерминированные функции δW₀ и M.
Степень как полинома от δW₀: ≤ 32 (из x² = x в GF(2)).

## Финальная структурная теорема

C = 2^128 потому что dim(output) = 256 = 128(e) + 128(a), ортогональны.

Четыре пути изменить 2^128:
1. Уменьшить выход — не атака
2. Нарушить ортогональность e/a — невозможно (T_DOUBLE_CHAIN)
3. Неслучайность выхода — мультипликативная (Oracle Distance 2^{-26})
4. Квантовый компьютер — не доступен

Экспонента n/2 — ИНВАРИАНТ выходного размера.
