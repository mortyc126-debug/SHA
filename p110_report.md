## КРИТИЧЕСКОЕ ИСПРАВЛЕНИЕ МЕТРИКИ

### Проблема П-104: неверный выбор регистра

```
В П-104 измеряли DEP только по b-регистру MD5.
b-регистр достигает оракула на r=2 (по одному регистру).
Это давало r*(MD5)=21 — артефакт медленного регистра a,c,d.

Правильная метрика: DEP по ВСЕМ регистрам состояния.
```

### Корректные значения (П-110, все регистры)

```
r    MD5(%all)   SHA-1(%all)   SHA-256(%all)
--   ---------   -----------   -------------
 1     2.45%        2.27%          2.10%
 5    63.02%       29.95%         88.15%  ← SHA-256 оракул
 7    95.14%       59.74%
 9    99.93%       86.60%
10    99.99%       93.97%         ← MD5 оракул
13             99.97%             ← SHA-1 оракул

r*(MD5,    все рег) = 7    →  G5 = 64/7  = 9.1
r*(SHA-1,  все рег) = 10   →  G5 = 80/10 = 8.0
r*(SHA-256,все рег) = 5    →  G5 = 64/5  = 12.8
```

### Все предыдущие числа r* пересмотрены

```
Из П-104:  r*(MD5)=21, r*(SHA-1)=21  ← НЕВЕРНО (один регистр)
Из П-110:  r*(MD5)= 7, r*(SHA-1)=10  ← ВЕРНО (все регистры)

Из П-109:  r*(SHA-256)=3  ← измерялось по e-регистру (частично верно)
Из П-110:  r*(SHA-256)=5  ← все регистры (более строгое)
```

## АНАЛИЗ: ЧТО G5 РЕАЛЬНО ИЗМЕРЯЕТ

### Почему MD5/SHA-1 взломаны при высоком G5?

```
G5 измеряет скорость диффузии случайного дифференциала.
Атаки Wang — это НЕ случайные дифференциалы.
Wang строит СПЕЦИАЛЬНО ПОДОБРАННЫЙ дифференциал
через sufficient conditions — избегая быстрой диффузии.

SHA-1 диффундирует за r*=10 для случайного δ.
Но Wang-атака строит дифференциал который "скользит"
по допустимому пути 80 раундов через specific bit conditions.
Это совершенно другой механизм.
```

### Новое понимание: G5 и структура атаки

```
G5 высокий (≥8) = случайный дифференциал быстро рассеивается.
Это НЕ означает что структурный дифференциал рассеивается.

Wang-атака обходит G5 через:
  - Sufficient conditions на биты промежуточных состояний
  - Специально построенный δW который не рассеивается
  - Это возможно потому что нелинейность MD5/SHA-1 СЛАБЕЕ

SHA-256 устойчив потому что:
  - Ch/Maj + три-ротационный S1 дают полную диффузию
    за ≤4 раунда ДАЖЕ для структурных дифференциалов
  - Wang-цепочка в SHA-256 тоже работает (П-92),
    но барьер r=17 — информационный, не структурный
```

### Что реально предсказывает G5

```
G5 предсказывает:
  ✓ Скорость диффузии случайного однобитового дифференциала
  ✓ Число раундов до оракульного поведения (любой хеш)
  ✓ Верхнюю оценку числа раундов Wang-цепочки

G5 НЕ предсказывает:
  ✗ Устойчивость к Wang-атакам (зависит от нелинейности функций раунда)
  ✗ Общую криптографическую стойкость
  ✗ Уязвимость к алгебраическим/SAT атакам
```
