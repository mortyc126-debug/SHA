# П-223: Три направления — итоги

## Направление A: Free-start Wang — ТРИВИАЛЬНО

**Результат**: 461K hits за 30с = P≈1.0

**Объяснение**: δIV[7]=-d0 → δT1[0]=0 → δe[1]=0 автоматически.
Затем адаптивный метод (W[1..15] свободны) гарантирует δe[2..16]=0 с P=1.
И расписание определяет ΔW[17]=0 тривиально.

**Вывод**: Это не Wang chain — адаптивные W[0..15] делают задачу тривиальной.
Настоящий Wang: W[0..15] фиксированы, и случайно δe[1..16]=0 с P≈2^{-104}.

## Направление B: 17-раундовый трейл — НАЙДЕН

**Трейл** (d0=8, NDR, 45M пар, 30с):
```
ΔW[0] = 0x00000008  HW=1  (d0)
ΔW[1] = 0x21000400  HW=3
ΔW[2] = 0xfffffff8  HW=29
ΔW[16]= 0xd7848822  HW=12
ΔW[17]= 0x00002000  HW=1  ← цель

δe[1] = 0x00000008  ✗ (≠0, это не Wang)
δe[2..16] = 0x00000000  ✓ (16 нулей)
δe[17] = 0x5eaa810f  ✗ (≠0, трейл заканчивается)
```

**Распределение HW(ΔW[17])** за 30с (~170M пар):
| HW | кол-во | % |
|----|--------|---|
| 2 | 16 | 0.00% |
| 3 | 193 | 0.00% |
| ... | ... | ... |
| 16 | ~24M | 14.00% |

Нет HW=0 и HW=1 встречается крайне редко (~1/45M пар).

**Важное ограничение**: Это адаптивный трейл (W[0..15] свободны).
Для настоящего SHA-256: W[16..63] = расписание от W[0..15].
В адаптивном трейле W[17] выбирается СВОБОДНО, что нарушает SHA-256 расписание.

**Что это означает**:
- Трейл валиден для СЖИМАЮЩЕЙ ФУНКЦИИ (free-start)
- Не является атакой на SHA-256 хеш-функцию

## Направление C: Branch&Bound

**Задача**: min HW(sig0(dw2) + dw1) при dw1 = -δS1(e0, d0) = 0x6affcc03

**Результаты**:
```
Линейный перебор (100M):   min HW=1  dw2=0x001a3c0d
Случайный поиск:           min HW=1  (подтвержден)
GF(2) анализ:              sig0 — линейная биекция
   sig0(dw2) XOR dw1 = 0  ← решение над GF(2) найдено
   sig0(dw2) + dw1 ≠ 0    ← carries мешают аддитивному нулю
```

**Важный вывод из C**:
- Над GF(2) задача решается точно (sig0 — биекция)
- Над Z_{2^32} с carries: carries создают разрыв между XOR и ADD
- HW=1 аддитивно (sig0(dw2)+dw1) достижимо за 100M итераций

## Что реально нового мы сделали

1. **NDR_Ch теорема**: d0=8 оптимален, δCh=0 → +15% для HW(ΔW[17])
