# П-209: Нейтральные биты, продление трейла, структура

## Исправление концепции нейтральных бит

Нейтральные биты (П-94/П-95) используются для РАЗНООБРАЗИЯ пар W,
но не для изменения ΔW[r] — при симметричном изменении W[r] и W'[r]
разность ΔW[r] не меняется, и ΔW[17] не меняется тоже.
Это объясняет почему P(wang)=100% в наших адаптивных трейлах.

---

## Структура адаптивного трейла (верифицирована)

```
r= 0: ΔW= 1бит (d0)
r= 1: ΔW=13бит (из S1 раунда 1)
r= 2: ΔW= 1бит (копия d0 — структурный факт!)
r= 3..15: ΔW≈16-28бит (случайные)
r=16: ΔW≈15бит (из расписания)
r=17: ΔW= 1бит ← РЕКОРД (П-207)
r=18..31: ΔW≈16бит (полная диффузия)
```

**δe профиль:**
```
δe[1] = d0 ≠ 0  (барьер SHA IV)
δe[2..16] = 0   (гарантировано адаптацией, 15 нулей)
δe[17..32] = 0  только 0/16 (полная диффузия после r=17)
```

**Итого: 15 δe=0 раундов + ΔW[17]=1бит ≈ 17 "дешёвых" раундов**

## Новые математические факты (П-209)

**T_DW2_STRUCTURAL**: ΔW[2] = d0 при ~50% трейлов (когда бит d0 установлен в e[2]).
При ΔW[2]=0: ΔW[17] = ΔW[1] → обычно ~13 бит.
При ΔW[2]=d0: ΔW[17] может быть мало из-за компенсации sig0(d0)+ΔW[1].

**T_DIFFUSION_WALL_R17**: За раундом 17 диффузия полная — δe[18..32]=0 не встречается.
Это подтверждает что "барьер ΔW[17]" является реальным препятствием для продления трейла.

**T_ADAPTIVE_17ROUNDS**: Адаптивный трейл даёт эффективный трейл длиной ~17 раундов:
- 15 раундов δe=0 (адаптация)
- +2 раунда "за счёт" ΔW[17]=1бит

## Следующий шаг П-210

**Двухблочная атака:**

Блок 1: раунды 0..17, W[0..15] → δe[2..16]=0, ΔW[17]=1бит
Блок 2: раунды 16..33, W'[0..15] → δe[18..32]=0 (отдельный адаптивный трейл)

Стыковка в середине: состояние после блока 1 == состояние для блока 2.
Это МИТМ (meet-in-the-middle) версия адаптивного трейла.

Если удастся — трейл покроет ~33 раунда против Li et al. рекорда 31.
