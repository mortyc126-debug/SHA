## T_BARRIER_SHIFT — главный результат

### Формулировка

При однобитовом дифференциале δW[j] = 1 (j = 0..15):

```
barrier(j) = j

где barrier(j) = max{ r : P(δe[r] = 0) = 1.0 }
```

**Полная таблица верификации:**

| j | barrier(δa) | barrier(δe) | Разница |
|---|---|---|---|
| 0 | 0 | 0 | 0 |
| 1 | 1 | 1 | 0 |
| 2 | 2 | 2 | 0 |
| ... | ... | ... | 0 |
| 15 | 15 | 15 | 0 |

Все 16 случаев: P=1.0, N=3000 пар каждый.

### Аналитическое доказательство (T_T1_GATEWAY)

Структура раунда SHA-256:

```
a[r+1] = T1[r] + T2[r]
e[r+1] = d[r]  + T1[r]
T1[r]  = h[r] + Σ1(e[r]) + Ch(e,f,g) + K[r] + W[r]
```

При δW[j] = 1 и δW[k] = 0 для k ≠ j:
- Раунды 0..j-1: W[r] не изменён → δT1[r] = 0 → δstate[r] = 0
- Раунд j: W[j] изменён → δT1[j] ≠ 0 → δa[j+1] ≠ 0 И δe[j+1] ≠ 0

**Ключевое наблюдение:** T1 входит одновременно в оба регистра a и e.
Поэтому δa и δe рождаются в одном раунде j — не с задержкой 4.

Ожидаемое по цепочке: barrier(δe) = j+4 (через a→b→c→d→e).
Реальное: barrier(δe) = j (через T1 напрямую).

## T_INTERFERENCE_MIN — закон интерференции

### Формулировка

```
barrier(δW[j1] ⊕ δW[j2]) = min(j1, j2)
barrier(δW[j1] ⊕ δW[j2] ⊕ δW[j3]) = min(j1, j2, j3)
```

Для любого набора индексов: барьер равен минимальному индексу.

### Следствие

Конструктивной интерференции не существует. Нельзя комбинацией
δW[j1]+δW[j2] получить барьер выше чем max(j1,j2).

Барьер 15 — абсолютный максимум достижимый через входные слова W[0..15].

## Новые принципы

**П-21:** barrier(δW[j]) = j — точный закон, нет исключений.
T1 является единственными воротами входа дифференциала в state.

**П-22:** Интерференция дифференциалов всегда деструктивна.
Комбинации не усиливают барьер — только разрушают.

**П-23:** δa и δe рождаются одновременно через T1, не через цепочку сдвига.
Модель "e = a сдвинутый на 4" неверна для дифференциального анализа.
