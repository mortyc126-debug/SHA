# Объединяющая алгебра SHA-256
# The Unifying Algebra

## Конструкция

R = Z₂[x]/(x³²-1) — кольцо 2-адических степенных рядов с циклической структурой.

Элементы: a = Σ aₖxᵏ, aₖ ∈ Z₂ (2-адические целые).

Операции SHA-256 в R:
- ADD: сложение в R, mod 2³² на коэффициентах
- ROTR^k: умножение на xᵏ
- XOR: сложение mod 2 (= редукция R → R/2R)

## Точная последовательность (F39 = нерасщепимость)

0 → 2R → R → R/2R → 0

R/2R = GF(2)[x]/(x³²-1) = "XOR мир"
2R = "carry мир" (ядро редукции)
Нерасщепимость: нет секции σ: R/2R → R сохраняющей кольцевую структуру.
Ext¹(R/2R, 2R) = F₂ — ровно один нетривиальный класс = carry.

## Линеаризация carry в Q₂

carry(a,b) = (a + b - a ⊕ b) / 2

В Q₂ (поле 2-адических дробей): деление на 2 возможно.
carry — ЛИНЕЙНАЯ функция в Q₂!

В Q₂[x]/(x³²-1):
- ADD: линейно ✓
- XOR: линейно ✓ (XOR = ADD - 2·carry, carry линеен)
- ROTR: линейно ✓ (умножение на x)
- Ch, Maj: НЕЛИНЕЙНЫ (побитовое AND = степень 2)

## SHA-256 в Q₂ = квадратичная система

В Z/2³²: SHA-256 collision = система степени 32 (carry)
В Q₂: SHA-256 collision = система степени 2 (только Ch/Maj)

Степень упала с 32 до 2!

## Цена: бесконечная точность

Q₂ = числа бесконечной 2-адической точности.
При редукции mod 2³²: carry возвращается.

## ОТКРЫТЫЙ ВОПРОС

Можно ли решить квадратичную систему в Q₂ и "округлить" до Z/2³²?
- Если да: collision cost = MQ complexity ≈ sub-exponential
- Если нет: carry восстанавливается при округлении

## Бимультипликативная алгебра

U = (R_carrier, +_R, ·_R, ·_S) — одно сложение, два умножения:
- ·_R = свёрточное (ROTR, schedule)
- ·_S = поточечное (Ch, Maj, AND)

Не кольцо (две операции умножения). Не near-ring.
Новый алгебраический объект.

## Carry как когомологический класс

[carry] ∈ Ext¹_R(R/2R, 2R) = F₂
Единственный нетривиальный класс расширения.
Уничтожается в Q₂ (Ext¹ = 0 для инъективных модулей).
