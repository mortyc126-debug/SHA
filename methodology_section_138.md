## ПОЛНЫЙ ИНСТРУМЕНТАРИЙ (итог всего исследования)

### Практические атаки

**[1] ECDSA Distinguisher (N=5-10 подписей)**
- Вход: подписи ECDSA без детерминированного nonce
- Выход: детектирование SHA-256 nonce с ~96% точностью
- Стоимость: минуты вычислений
- Путь: score → HNP → LLL → приватный ключ

**[2] SHA-16 Birthday (частичная коллизия)**
- Вход: ~2400 SHA-16 carry=0 примеров
- Выход: коллизия H[7]^H[6] (64 бит)
- Стоимость: секунды
- Ограничение: частичная, не полная коллизия

### Академические результаты

**[3] Universal SHA-r Distinguisher**
- AUC ≥ 0.60 для любого r=8..64
- Первая систематическая атака на семейство SHA-r
- Аналитические инварианты через K[r-1]

**[4] Детерминированные инварианты SHA-256**

При carry[63]=0 строго выполняется:
```
Ch[62][b31]=0, Ch[62][b30]=0   (через K[63], доказано)
g[62][b31]=0,  g[62][b30]=0    (через SS<2^29.85)
f[61][b31]=0,  f[61][b30]=0    (g[62]=f[61])
e[60][b31]=0,  e[60][b30]=0    (горлышко)
```
