# П-208: SC анализ — два класса трейлов

## Исходная пара П-207

```
W0    = 0x7be6d516  d0 = 0x00000008 (бит 3)
ΔW[1] = 0x910023ef  HW=13
ΔW[2] = 0x00000008  HW=1  ← копия d0!
ΔW[17]= 0x00000080  HW=1
```

---

## Структурные факты (верифицированы)

**T_DW2_EQUALS_D0**: при адаптивном трейле ΔW[2] = d0 (точная копия).

Доказательство: раунд 2 correction = δCh(e[2], f[2], g[2]).
При δe[2]=0, δg[2]=0, δf[2]=δe[1]=d0:
```
δCh = Ch(e, f+d0, g) - Ch(e, f, g) = e & d0
```
Если бит d0 не установлен в e[2] → correction=0 → ΔW[2]=0.
Если установлен → ΔW[2]=d0. Вероятность 50/50.

**T_DE1_BARRIER**: δe[1] = d0 ≠ 0 при SHA IV — тождественно.
```
δe[1] = δd[0] + δT1[0] = 0 + (δh[0]+δS1(e[0])+δCh+d0) = 0+0+0+d0 = d0
```
Это не преодолеть в стандартной SHA-256 модели (оба трейла стартуют с SHA IV).

---

## Два класса трейлов

| Класс | δe[1] | HW(ΔW[17]) | Модель |
|-------|-------|------------|--------|
| Адаптивный (SHA IV) | d0 ≠ 0 | **1** | Partial Wang |
| Free-start (δIV[7]=-d0) | **0** | **0** | Wang-совместимый |
| Настоящий Wang chain | **0** | 3 (числово) | P≈2^{-104} |

---

## Free-start результат (новый!)

**Конструкция**: IV₁ = SHA_IV, IV₂ = SHA_IV + (0,0,0,0,0,0,0,-d0)

```
δIV[7] = -d0 = 0xfffffff8
Тогда: δT1[0] = δh[0] + d0 = -d0 + d0 = 0
→ δe[1] = 0 ✓
```

**Результат**: HW(ΔW[17]) = 0 с первой пары! (P=100%, тривиально)

Free-start адаптивный трейл имеет δe[1..16]=0 и ΔW[17]=0 — это
идеальный трейл, но в ослабленной модели (IV не фиксирован).

---

## Иерархия трейлов (П-208)

```
Free-start адаптивный:  δe[1..16]=0, ΔW[17]=0  ← тривиально (δIV компенсирует)
          ↓ ужесточение: фиксируем IV = SHA_IV
SHA IV адаптивный:      δe[2..16]=0, ΔW[17]=1  ← П-207 результат
          ↓ ужесточение: W[1..15] не адаптируются
Настоящий Wang chain:   δe[1..16]=0 (требует SC), ΔW[17]≥3
```

## Открытый вопрос для П-209

**Гибридная атака**: начать с Wang chain пары из П-97 (W0=0xc97624c6),
и использовать нейтральные биты W[12,13,15] (96 бит свободны),
чтобы минимизировать HW(ΔW[17]).

Ожидаемый результат: из 2^96 вариантов нейтральных бит,
найдётся вариант с малым HW(ΔW[17]).
Если Wang pair имеет HW(ΔW[17])≈16 в среднем,
то среди 2^96 вариантов минимум будет ≈ 0 бит.

**Это реальная задача без Gurobi.**
