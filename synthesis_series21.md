# Серия XXI: Синтез трёх направлений

## Что нашли в этом сеансе

### Направление A: Новый дифференциальный путь

**Ключевое открытие:** наша MILP-модель искала детерминистический путь (δstate=0 везде). Это неправильная постановка. Правильная задача — вероятностный путь (как у Li-Liu-Wang).

**Что нужно строить:**
- MILP с вероятностями переходов XDP/ADP для MODADD
- Наши теоремы (T_BACKWARD_PROPAGATION, T_DIFFUSION_DECOMP) дают новые ограничения которых нет в Li-Liu-Wang
- Потенциал: расширить 31R → 40-50R

**Сложность:** высокая. Требует глубокого понимания XDP/ADP.

---

### Направление B: Chosen-prefix коллизия

**Ключевое открытие:**
```
T_WANG_IV_INDEPENDENCE: Wang cascade работает для любого IV.
Доказательство: δe[1] = DW[0] для любого IV (T_FIRST_STATE_FORCED).
Следствие: SHA-256 chosen-prefix стоит O(2^32).
```

**Что верифицировано:**
- δe[1] = 0x80000000 для стандартного, нулевого и случайного IV ✓
- T_HW64_INDEPENDENCE работает для любого IV ✓

**Риск:** это может быть folklore. Нужна литературная проверка.

**Если не folklore → публикация уровня FSE/ASIACRYPT.**

---

### Направление C: Нижние границы

**Два строгих результата:**

```
T_LOWER_BOUND_SCHEDULE (уже доказана, Серия XVIII):
  C ≥ 2^32 для любой атаки с deterministic SC.
  Tight: достигается Wang chain.

T_GAP_THEOREM (частично):
  Классы атак:
    Структурные (diff, Wang-style): C ≥ 2^32
    Random oracle: C ≥ 2^128
  В зазоре (2^32, 2^128): публично ничего не известно.
```

**Частичный результат (реалистичный):** T_LOWER_BOUND_SCHEDULE → ToSC/FSE.

**Полная Gap Theorem:** CRYPTO Best Paper. Многолетняя работа.
