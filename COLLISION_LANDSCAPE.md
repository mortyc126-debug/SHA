# Ландшафт коллизий SHA-256

## Коллизии = изолированные точки в плоской пустыне

L_M(δM) = HW(H(M) ⊕ H(M⊕δM)). Коллизия: L = 0.

Avalanche: L(δM* ⊕ e_i) ≈ 128 для всех единичных e_i.
Radius долины = 0. Нет градиента. Нет склона.

## Кластеризация коллизий

~2^24 коллизий на расстоянии ≤ 56 от каждой (Gilbert-Varshamov).
Переход = стандартный birthday 2^128. Кластеры не помогают.

## Плотность локальных минимумов

Σ(E) = S_L(E). Каждая near-collision = локальный минимум.
Ландшафт полностью плоский с микроскопическими дырками.

## Collision correlation

ρ(M₁, M₂) ≈ 2^{-256}. Нет универсальных дифференциалов.

## Glass transition

T_g ≈ 0.33 (min L ≈ 84 для SA). Алгоритмический барьер, не структурный.

## Вывод

Ландшафт SHA-256 = ландшафт случайной функции.
Нет эксплуатируемой структуры для local search.
