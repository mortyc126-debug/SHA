## EXP A: Wang-MD5 vs Wang-SHA256

```
Свойство        Wang-MD5              Wang-SHA256
Метод           Sufficient cond.      Newton адаптация
P(цепочки)      < 1 (вероятн.)       = 1.0 (детерм.) ← сильнее!
Мёртвые раунды  Используются         Используются ✓
Нулей           64/64 (вероятн.)     16/17 (детерм.)
Барьер          нет (→коллизия)      r=17 (P=2^-32)
Нейтр. биты     да (Wang 2004+)      да (96 битов) ✓

Верификация: Wang-SHA256 P=1.0: 1000/1000 ✓
```

Wang-SHA256 строго детерминированнее Wang-MD5, но ограничен
барьером r=17. Это структурно новый результат.

## EXP C: Нейтральные биты → δe[18]=0

```
Эксперимент: birthday в Wn[13] при δe[17]=0.
2 000 000 итераций, 350k/s:
  P(δe[18]=0 | δe[17]=0, vary Wn[13]) = 0/2M = 2^-32

Нейтральные биты НЕ снижают стоимость поиска δe[18]=0.
Каждый новый ноль стоит отдельный birthday O(2^32).
```
