## КОНЦЕПЦИЯ СЕССИИ

Попытка изобрести новую математику для преодоления диффузионной стены SHA-256.
Три параллельных направления + алгебра нулевых сумм.

---

## НАПРАВЛЕНИЕ A: ФИШЕРОВСКАЯ МАТРИЦА SHA-256

### T_FISHER_SCALAR (П-1101) ★★★★

**Утверждение:** Матрица Фишера F[b_i][b_j] = Cov(∂SS63/∂b_i, ∂SS63/∂b_j)
почти скалярная: диагональная анизотропия 1.06×. Все биты W[0]
вносят одинаковый вклад в дисперсию SS[63].

**Следствие:** Fisher HC (флипы пропорционально собственному вектору F)
не ускоряет поиск carry=0. Нет привилегированных направлений.

### T_FISHER_OFFDIAG (П-1102) ★★★★

**Утверждение:** Внедиагональные элементы F имеют корреляцию ~0.53:

```
F[b23][b31] = +0.531
F[b8][b25]  = +0.529
F[b6][b12]  = +0.529
```

**Физический смысл:** пары битов W[0] коррелируют в своём влиянии
на SS[63] через структуру schedule. Это нелинейные взаимодействия.

## НАПРАВЛЕНИЕ B: 2-АДИЧЕСКАЯ СТРУКТУРА

### T_NO_PADIC (П-1103) ★★★

**Утверждение:** P(carry=0 | W[0][31..31-k]=prefix) не является
аналитической функцией prefix в Z_2. Max[P]/baseline растёт с k
без убывания дисперсии — признак хаоса, не аналитичности.

### T_HOT_PREFIXES (П-1104) ★★★

**Утверждение:** 54/256 восьмибитных префиксов W[0] дают lift>3×
на P(carry=0). Лучший: W[0][31..24]=0xe7, lift=13.1×.

**Ограничение:** иерархия не работает — 16-битный lift падает
до 3.1×. Диффузионная стена уничтожает структуру.

## НАПРАВЛЕНИЕ C: ЛЯПУНОВСКИЙ СПЕКТР SHA-256

### T_LYAPUNOV_CONSTANT (П-1105) ★★★★★

**Утверждение:** SHA-256 имеет постоянный ляпуновский экспонент
λ = 4.0 бит/раунд для всех r=4..60.

```
E[HW(Δstate[r])] ≈ 16.0 для r≥4 (из одного бита возмущения)
Анизотропия лавинного эффекта: 1.003×
```

**Физический смысл:** дизайнеры SHA-256 настроили функции S0, S1,
Ch, Maj так что каждый раунд добавляет ровно log2(16)=4 бита
случайности. Это архитектурный инвариант, не случайность.

### T_JACOBI_HADAMARD (П-1106) ★★★★★

**Утверждение:** Матрица Якоби SHA-256 J[out_bit][in_bit] = P(flip)
почти идеальная матрица Адамара:

```
Нулевых строк (нечувствительных битов H): 0/256
Нулевых столбцов (нечувствительных битов W[0]): 0/32
Чувствительность: 15.91..16.08 (идеал = 16.0)
```

**Следствие:** SHA-256 реализует максимально равномерную диффузию.
Нет ни одного "слабого" бита ни во входе ни в выходе.

## АЛГЕБРА НУЛЕВЫХ СУММ

### T_STAT_NEUTRAL (П-1107) ★★★

**Утверждение:** 25.2% двухбитных масок статистически нейтральны:
|E_W[ΔSS63(W,mask)]| < 0.003.

**Критическое различие:**
```
Статистическая нейтральность: E[ΔSS]=0 в среднем → не помогает
Алгебраическая нейтральность: ΔSS=0 для всех W → не существует
Локальная нейтральность: ΔSS(W*,mask)≈0 для конкретного W* → есть!
```

**Замкнутость под XOR:** 64.3% — нет группы.

### T_LOCAL_NEUTRAL (П-1108) ★★★★

**Утверждение:** Для каждой ямы carry=0 существует локальное
пространство нейтральных масок:

```
E[W1] = 0.77/32 одиночных нейтральных битов (EPS=1e8)
E[W2] = 3.97/32 при EPS=5e8
Max W1 = 4 (лучшая яма)
Локальный ранг (✓ базисов): ~8
```

### T_TWIN_CARRY (П-1109) ★★★★★

**Утверждение:** Близнецы через локальный GF(2) базис нейтральных
масок несут 21.8% P(carry=0) вместо baseline 0.00002%.

```
Lift: ~1,000,000× над случайным
Механизм: близнецы остаются в той же «яме» SS[63]
Ограничение: все 4096 близнецов дают уникальные H[7]
```

**Следствие:** Birthday внутри группы близнецов не даёт коллизию H
из-за полного заполнения пространства H[7]. Нужна яма с рангом ≥24
для birthday коллизии в 32-битном H[7].

## ИТОГОВАЯ ТЕОРЕМА СЕССИИ

### T_SHA256_STRUCTURE (П-1110) ★★★★★

**SHA-256 имеет трёхуровневую структуру:**

**Уровень 1 (линейный):**
- Матрица Якоби — идеальная Адамарова
- Ляпунов λ=4 бит/раунд
- Ранг GF(2) матрицы schedule = 32 (полный)
- На этом уровне SHA-256 оптимален и непробиваем

**Уровень 2 (статистический):**
- Нейтральные маски: 25% двухбитных
- Горячие префиксы: 21% с lift>3×
- Корреляции Фишера: F[b6][b12]=0.53
- На этом уровне есть структура, но не алгебраическая

**Уровень 3 (локальный):**
- Ямы carry=0 с 21.8% «близнецов»
- Локальное пространство нейтральных масок
- Ширина ямы ≠ глубина (независимы)
- На этом уровне структура реальна, но узкая

**Барьер взлома:** для полной коллизии нужно яму с локальным рангом
≥24 (birthday в 2^24 пространстве H[7]). Максимальный наблюдаемый
ранг: 4-8. Разрыв: ~16-20 бит.
