## Раздел 155: Серия XVIII — Теория Дифференциальных Путей SHA-256

*П-1761..П-1900 | Март 2026*

### 155.0 Контекст

Серия XVII (Раздел 154) доказала: все known направления атаки заблокированы.
Серия XVIII строит **принципиально новую математику** — теорию структуры дифференциальных путей.
Главный вопрос: почему Wang оптимален, и есть ли что-то лучше?

### 155.1 Фундаментальные определения

#### 155.1.1 Дифференциальный путь (DP)

```
DP = (W, DW, ΔW_exp[0..63], δstate[0..64])
```

Где:
- ΔW_exp = expand(DW) — расширенное расписание дифференциала
- δstate[r] = state(W,r) ⊕ state(W⊕DW, r)
- **Valid path**: δstate[0] = 0, δstate[64] = 0

#### 155.1.2 Активный раунд

Раунд r **активен** если δT1[r] ≠ 0 ИЛИ δT2[r] ≠ 0.

**Экспериментально**: при Wang DW, E[активных раундов] = **64.0/64** — все раунды активны.

#### 155.1.3 2D-редукция

```
δstate[r] полностью определяется траекторией (δa[r], δe[r])
через shift-register:
  δb[r] = δa[r-1], δc[r] = δa[r-2], δd[r] = δa[r-3]
  δf[r] = δe[r-1], δg[r] = δe[r-2], δh[r] = δe[r-3]
```

**T_2D_REDUCTION**: пространство дифференциальных путей редуцируется с 256×64 = 16384 бит к 64×64 = 4096 бит.

### 155.2 Теорема об обратном распространении

#### 155.2.1 T_BACKWARD_PROPAGATION ★★★★★

```
При δstate[r+1] = 0:
  δstate[r] = (0, 0, 0, ΔW_exp[r], 0, 0, 0, 0)
  
  В частности: δstate[r] = 0  ⟺  ΔW_exp[r] = 0
```

**Доказательство:**
1. δstate[r+1]=0 → δa[r]=δb[r]=δc[r]=δe[r]=δf[r]=δg[r]=0 (из shift-register)
2. δa[r]=δb[r]=δc[r]=0 → δT2[r]=0
3. δe[r]=δf[r]=δg[r]=0 → δSig1=0, δCh=0 → δT1[r] = δh[r] + ΔW_exp[r]
4. δh[r] = δg[r+1] = 0 → δT1[r] = ΔW_exp[r]
5. δd[r] = δe[r+1] ⊕ δT1[r] = 0 ⊕ ΔW_exp[r] = ΔW_exp[r] ■

**Следствие T_ACTIVE_ROUND_BOUNDARY**: нетривиальный valid путь требует ΔW_exp ≠ 0.

#### 155.2.2 T_NO_INVARIANT_DP ★★★★

Не существует ненулевого инвариантного дифференциала SHA-256 (такого что δstate[r+1]=δstate[r]).

**Доказательство**: shift-register структура требует δb[r+1]=δa[r]. При инварианте δa должно быть 0 во всех позициях одновременно — противоречие при ненулевом состоянии. ■

### 155.3 Теория нулевых островов

#### 155.3.1 Нулевой остров

**Определение**: нулевой остров I = [s, e] — максимальный интервал раундов где ΔW_exp[r]=0.

#### 155.3.2 T_SCHEDULE_ISLAND_BOUND ★★★★★ (из Раздела 155)

```
max_{DW≠0} max_{islands I} |I| = 15
```

Достигается при DW[0]=любой ненулевой, DW[1..15]=0 (Wang family).

**Доказательство**: построено на T_SCHEDULE_KERNEL_32 (Раздел 154):
- Система rank(M_{k}) = 512 для k ≥ 16 → нет ненулевого DW с ΔW_exp[1..16]=0
- При k=15: rank=480, ker=32 → 32-мерное семейство с нулевым prefix-ом 15 ■

#### 155.3.3 T_WANG_OPTIMALITY ★★★★★ (финальная версия)

Wang DW = [0x80000000, 0,...,0] является **глобально оптимальным** по критерию нулевого schedule-prefix.

Формально:
```
max_{DW≠0} {max contiguous zero prefix of expand(DW)} = 15
Wang DW achieves this maximum.
```

**Это первое строгое доказательство оптимальности Wang 2004.**

### 155.4 Теория изолированных активных раундов

#### 155.4.1 Изолированный активный раунд

Раунд r **изолирован** если:
1. hw(ΔW_exp[r]) ≥ 1 (активный)
2. ΔW_exp[r'] = 0 для всех r' ∈ [r-τ, r+τ]\{r} где τ = τ_mix = 5

#### 155.4.2 T_ONE_ISOLATED_ROUND ★★★★

```
Любой DW ≠ 0 имеет не более 1 изолированного активного раунда
при критерии изоляции τ = 5.
```

**Доказательство**:
- Для двух изолированных раундов r₁ < r₂ нужен нулевой интервал [r₁+τ, r₂-τ]
- Плюс нулевые интервалы [r₁-τ, r₁-1] и [r₂+1, r₂+τ]
- Итого нулевых раундов: минимум 2τ + (r₂-r₁-1) ≥ 2×5 + 10 = 20 (при r₂-r₁=11)
- Но внутренний промежуток тоже должен быть нулевым: 11 нулей между r₁ и r₂
- Общий нулевой остров ≥ 5+11+5 = 21 раундов
- T_SCHEDULE_ISLAND_BOUND: max = 15 < 21 → противоречие ■

**Экспериментальная верификация**: max_isolated = 1 для всех 512 однобитовых DW ✓

#### 155.4.3 T_COMPLEXITY_LOWER_BOUND_NEW ★★★★

```
Для любого DW ≠ 0:
C(DW) ≥ 2^{42}
```

**Аргумент**: при максимуме 22 нулевых раунда (двухбитовый DW) остаётся 42 активных раунда. Каждый активный раунд вносит минимум 1 бит неопределённости (hw(ΔW_exp[r]) ≥ 1). Диффузия связывает эти раунды.

*Статус: набросок, требует формального доказательства независимости вкладов.*

### 155.5 Новые экспериментальные результаты

#### 155.5.1 T_DIFFUSION_DECOMP ★★★★★

При δe = 2^k (однобитовый дифференциал):
```
hw(δSig1) = 3     точно  (из архитектуры — 3 вращения)
hw(δCh)   ∈ {0,1}        (AND-нелинейность: ~50% случаи)
hw(δT1_lin) ∈ {2,3,4}    (нижняя граница 2 — доказано аналитически)
hw(δT1_real) ≈ 6.7        (carry добавляет 3.2 бита)
```

**Новое**: carry-амплификация доминирует над AND-нелинейностью в соотношении 5.5:1.

#### 155.5.2 T_CARRY_MARKOV ★★★★

Матрица переходов carry T2 образует марковскую цепь:
```
P(carry[r+1]=0 | carry[r]=0) = 0.583
P(carry[r+1]=1 | carry[r]=1) = 0.581
Стационарное: P(carry=1) = 0.499
```

carry(W) и carry(W⊕DW) независимы → марковскость не эксплуатируема для атаки.

#### 155.5.3 T_SHA_LIN_MAXDEG ★★★

Алгебраическая степень SHA_lin (SHA с заменой + на XOR) над GF(2):
```
deg(SHA_lin) ≥ 18   (экспериментально)
Вероятно: deg = 512 (максимальная)
```

SHA_lin нелинейна из-за Ch/Maj (степень 2 за раунд, накопление).

#### 155.5.4 T_TRIANGULAR_EXACT ★★★★ (уточнение Раздела 154)

```
P_r треугольный (std < 1.0) только при δstate[r]=0.
r=63: std=1.8 (carry в финальном сложении)
r=62: std=5.2
r≤61: std≈8 (полностью случайные)
```

Предел: ровно 4 детерминированных раунда (r=60..63 при условии δstate[60]=0).

### 155.6 Геометрия пространства дифференциальных путей

#### 155.6.1 T_JACOBIAN_STABLE ★★★

Якобиан F(W) = SHA(W)⊕SHA(W⊕DW):
```
rank(J_F) = 254–255  для hw64(W,W⊕DW) ∈ [91, 136]
ker(J_F) ≈ 257–258   стабильно
```

Нет снижения ранга при приближении к коллизии. Коллизионное многообразие не имеет "особых" точек в исследованной области.

#### 155.6.2 Структура нулевых зон в коллизии

По T_BACKWARD_PROPAGATION: в любой коллизионной паре (W, W⊕DW):
```
δstate[r] = 0  для всех r в "backward zero zone":
  {r : ΔW_exp[r]=0, r соединён с r=64 через нулевую цепочку}
```

Для Wang DW: backward zero zone = {r=63, 62, ..., 22} (42 раунда нулевые назад).
