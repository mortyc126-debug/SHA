# П-77: Четыре новых направления — первые результаты (Март 2026)

## Направление 5: Бит-гранулярная диффузия ★★★ ГЛАВНОЕ ОТКРЫТИЕ

### Новая теорема T_BIT_LINEAR_R1:

```
Раунд 1 SHA-256 — чисто линейный побитовый перенос:
  e_1[b] = W[0][b]  для всех b = 0..31.
  Вероятность: P = 1.0 (детерминировано).

Следствие: дифференциальный путь с активным W[0]
в раунде 1 имеет ТОЧНО предсказуемый вид в e_1.
Это устраняет неопределённость первого раунда.
```

### Медленные пути (P < 0.001):
```
W[1] бит 14 → e_3 бит  0: P = 0.0003
W[1] бит 14 → e_3 бит  1: P = 0.0003
W[1] бит 16 → e_3 бит  1: P = 0.0003
...
```
Это в 100× медленнее случайного ожидания (P ≈ 0.5).
Биты 0,1,2 регистра e_3 — наименее связанные с входом.

### Применение:
Строить дифференциальный путь через «медленные» биты:
вместо слов (32 бита) работать с конкретными битами где диффузия слабее.

## Направление 3: Schedule как линейный код ★★★

### Результат:
- `rank(G) = 512` — полный ранг. Нет глобального ядра.
- Пара `W[8]бит0 + W[8]бит1` → **7 нулей** из 48 в расширенном расписании.
- 7/48 = 14.6% нулей при HW=2. **Структурный, не случайный эффект.**

### Новая теорема T_SCHEDULE_CLUSTER:
```
Существуют пары бит (b1, b2) ∈ W_input такие что
|{k : W_expanded[k][b'] = 0}| >> E[случайно]
для соответствующих выходных битов b'.

Пример: W[8]бит0 ⊕ W[8]бит1 → 7 нулей в W_expanded[16..63].
```

### Следующий шаг:
Искать HW=3,4 пары с 15+ нулями → основа для sparse дифференциального пути
где расписание имеет мало активных слов.

## Направление 2: Carry Skeleton ★★

### Результат:
- `streak=4` (4 раунда carry-free подряд) найден за 307K итераций.
- Это быстрее ожидания для независимых событий (1M).
- Корреляция между carry-free состояниями соседних раундов: **есть**.

### Гипотеза T_CARRY_CORRELATION:
```
P(e_{r+1} carry-free | e_r carry-free) > P(e_{r+1} carry-free)
То есть carry-free состояния НЕ независимы по раундам.
Следствие: существуют «carry-free траектории» длиннее случайных.
```
Требует строгой проверки: измерить P(streak=k)/P_random(streak=k).
