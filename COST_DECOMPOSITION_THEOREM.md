# Финальная теорема: Декомпозиция стоимости коллизии SHA-256

## Теорема

Стоимость коллизии SHA-256: C = 2^{n_e/2} × 2^{n_a/2} = 2^{n/2} = 2^128

где n_e = 128 (бит уравнений e-ветви), n_a = 128 (бит уравнений a-ветви).

## Доказательство

1. Wang chain: δe[2..16] = 0 за O(1), 16 свободных W
2. Барьер: δe[17..20] = 0 — 128 бит. Birthday: 2^64
3. Сдвиговый регистр: δ(f,g,h)[20..23] = 0 автоматически
4. Остаток: δa[17..20] = 0 — 128 бит. Birthday: 2^64
5. Ортогональность (T_DA17_DE17_ORTHOGONAL): δe и δa независимы
6. Итого: 2^64 × 2^64 = 2^128

## Почему ортогональны

δe_new = δd + δT1. Фиксация δe = 0 → δT1 = -δd (определено).
δa_new = δT1 + δT2 = -δd + δT2. Зависит от δT2 = Σ₀(δa_prev) + Maj.
δT2 не зависит от текущего δe → ортогональны.

## Путь к sub-birthday

Нарушить ортогональность: найти carry-корреляцию δe ↔ δa.
Carry в T1 — общий, но фиксация δe определяет T1 → δa зависит только от T2.
Корреляция = 0 (подтверждено).

## Ретракция Q₂

carry(a,b) = (a+b - a⊕b)/2 — тавтология (содержит ⊕).
XOR не полиномиальна в Q₂. Степень carry = 33 в любом представлении.

## Что верно из объединяющей алгебры

- Кольцо R = Z₂[x]/(x³²-1): корректный фреймворк ✓
- Точная последовательность 0 → 2R → R → R/2R → 0: нерасщепима ✓
- Ext¹ = F₂: carry = единственный нетривиальный класс ✓
- Бимультипликативная алгебра U: новый объект ✓
