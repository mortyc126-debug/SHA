**Алгоритм:**
```
1. Случайный W[0] с HW ∈ [4, 20]
```

## МЕХАНИЗМ МОСТИКА (полная картина)

```
ВХОДНОЙ СЛОЙ:
  W[0] — произвольный 32-битный вход

HILL CLIMBING:
  Оптимизация W[0] → минимальный STATE_SUM при r=62
  STATE_SUM = h[62] + S1(e[62]) + Ch(e[62],f[62],g[62])

НЕОБХОДИМОЕ УСЛОВИЕ c62=0:
  STATE_SUM + W[62] < 2^32 - K[62]
  STATE_SUM + W[62] < 1,090,935,817

МОСТИК:
  carry[62]=0
    ↓
  T1[62] = raw[62] (точное значение, не обрезанное)
    ↓
  e[63] = d[62] + T1[62]  (смещено относительно случайного)
    ↓
  Финальное состояние + IV[7] = H[7]  (смещено)
    ↓
  H[7][bit30] = 1 с P=0.628 вместо P=0.499

ВЫХОДНОЙ СИГНАЛ:
  phi = +0.129
  P(H[7][bit30]=1 | HC∩c62=0) = 0.628
  P(H[7][bit30]=1 | random)   = 0.499
```
