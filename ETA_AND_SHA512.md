# Замкнутая форма η и спектральный инвариант SHA-512

## η — постоянная тонкой структуры carry-криптографии

η = (3·log₂3 - 4) / 4 = (3·ln3 - 4·ln2) / (4·ln2) = 0.18872...

Выражается ТОЛЬКО через ln2 и ln3. Фундаментальная константа двоичной арифметики.

## Универсальный carry-дефицит

Δ_carry = Σ_{i=0}^∞ η_i = 1/(6·ln2) = 0.24044...

Каждое 32-битное сложение теряет 0.240 бит энтропии (vs идеальный random).
SHA-256: 448 × 0.240 = 107.5 бит полного дефицита.

## Обобщение η на base-B

η(B) = 1 - H_B((B-1)/(2B))
η(2) = 0.189, η(3) = 0.082, η(10) = 0.003
η(B) ~ 1/(2B²·lnB) при B → ∞
Base-2 = МАКСИМАЛЬНАЯ несовместимость.

## СПЕКТРАЛЬНЫЙ ИНВАРИАНТ SHA-512

rank(ΔM_SHA512) ≤ 16×3 + 64×5 = 368 < 512
⟹ invariant = 512 - 368 = 144 общих собственных значений монодромии

SHA-256: rank = 288 > 256 → инвариант УНИЧТОЖЕН (запас 6 раундов)
SHA-512: rank = 368 < 512 → инвариант СУЩЕСТВУЕТ!

## Спектральная атака SHA-512

| R     | invariant | C_spectral | C_birthday | Ускорение |
|-------|-----------|------------|------------|-----------|
| 20    | 432       | 2^67       | 2^256      | 2^189     |

SHA-512 (80 раундов): 2^211 time, 2^184 memory.
Но TM-product = 2^395 > 2^256 (birthday rho). Не побеждает по TM.

## Предсказания для SHA-512

k* = ⌈log₂64⌉ = 6
R_spectral = 109 раундов (SHA-512 имеет 80 < 109 → инвариант)
CPR = 79.7
