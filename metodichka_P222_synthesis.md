# П-222: Синтез + NDR_Ch оптимизация

## Что мы сделали: синтезировали знания П-1..П-221

### Новая идея: NDR_Ch целенаправленный d0

Из аналитики П-198:
```
δCh(e,f,g,d0) = d0 & (f⊕g)
При d0 & (f⊕g) = 0: δCh=0 гарантировано
```

SHA IV: f[0]=0x9b05688c, g[0]=0x1f83d9ab
```
f⊕g = 0x84864327  (HW=12)
NDR_Ch маска = ~(f⊕g) = 0x7b79bcd8  (HW=20 свободных бит)
```

### Результат

| Тип d0 | avg min HW(ΔW[17]) | улучшение |
|--------|-------------------|-----------|
| NDR (δCh=0) | **3.21** | +15% |
| не-NDR | 3.85 | — |

**15% улучшение** от использования знания о SHA-256 структуре!

### 64-раундовый трейл

Лучший d0=0x00000008 (NDR, бит 3):
```
min HW(δstate[64]) = 34  за 5M пар
(vs 36 за 5M пар в П-219b при d0=8)
```

Небольшое улучшение, но оно есть.

## Синтез: что работает, что нет, что дальше

### Работает (верифицировано)
- NDR_Ch выбор d0: +15% для HW(ΔW[17])
- 4-блочный трейл: min HW=30 за 200M пар
- T_MULTIBLOCK: автоматическое продление
- Структура δstate: (δa,δb,δc,δd,0,0,0,0)

### Не работает
- MILP без Gurobi: тривиальные решения
- МИТМ < O(2^64)
- N-блочный → HW не убывает
- Wang chain без birthday (~2^104)

### Честная оценка барьера

Фундаментальная проблема:
```
Адаптивный трейл: W[0..63] свободны → мощный, но не SHA-256
SHA-256 атака: W[0..15] фиксированы → расписание ограничивает
```

Для реальной атаки на SHA-256 (стандарт, один блок):
- Нужен Wang chain: δe[1..16]=0, P≈2^{-104}
- Или радикально новая структура

### Следующий реальный шаг без Gurobi

**Вариант 1: Улучшенный Wang birthday**
Из П-97: Wang pair найден за 8685M итераций (~5 мин CPU).
С нашими знаниями о NDR_Ch: выбираем d0=0x8 (NDR),
ищем W[0..15] с δe[1..16]=0 + δCh=0 в раунде 1.
Это добавляет constraint → вероятно повышает P чуть выше 2^{-104}.

**Вариант 2: Быстрый Wang с GPU**
Код из П-97 (C, 2.1M/с) → GPU даст 200M/с → Wang за секунды.
После нахождения Wang pair: применяем NDR оптимизацию для ΔW[17].

**Вариант 3: Продление к 17+ раундам**
Wang chain (16 раундов) + один адаптивный раунд (17).
При ΔW[17]=1 бит: эффективный трейл 17 раундов.
Это превышает многие академические результаты.
