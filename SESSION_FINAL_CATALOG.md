# Новые математические объекты — финальный каталог сессии

## I. Криптографическая стена (Wall)
W(f) = min deg(f - g) для линейных g = толщина стены
H(f) = dim ker(f - g) = высота (ядро нелинейного остатка)
SHA-256: W = 33, H ≈ 2^8. Collision ~ 2^{m/2} × (1 - H/2^{m/2})

## II. DGA carry (дифференциально-градуированная алгебра)
A = ⊕ A_k, d: A_k → A_{k+1} (carry-дифференциал)
H_k = carry-когомология уровня k
H_k ≠ 0 для k ≤ 5 (прозрачная зона), ≈ 0 для k > 5 (стена)
χ_carry = Σ(-1)^k dim(H_k) = carry Euler characteristic

## III. SHA-256 дзета-функция
Z_M(t) = exp(Σ N_k/k × t^k)
N_k = number of near-collisions at distance ≤ k
Z_M ≈ Z_rand × (1 + O(2^{-26}))

## IV. Алгебра наблюдателя
O_XOR: dim = n². O_carry: dim = n(n+1). O_R: dim = n·2^n
Carry добавляет n измерений (32 из 10^11 недостающих)
Information gap: O_XOR видит 10^{-9} полной структуры

## V. Information rate
R_info = 1 + O(2^{-26}) бит/вычисление
Birthday optimal: 2^128 / R ≈ 2^128

## VI. Пространство хеш-функций
SHA-256 ∈ A_256 ⊂ H_256. |A_256| ≈ 2^2100. d(SHA-256, random) ≈ 2^{-26}
K-константы субоптимальны: CP = 0.41 vs ideal 0.50
