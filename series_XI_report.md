## 3. НОВЫЕ ТЕОРЕМЫ

### T_CARRY_NULL
```
Для случайного W ∈ {0,1}^512:
  φ(carry[i], carry[j]) = 0  (все пары i,j ∈ Zone-2)
  
Доказательство: эмпирически, N=50,000, |φ|<0.003 << σ=1/√N=0.0045
```

**Следствие:** carry-сигналы из Zone-2 сессии были артефактом обусловленной выборки (Wang-pairs). Они реальны — но только ВНУТРИ Wang-пространства. Для неконтролируемого SA они не применимы как независимые guides.

### T_CARRY_CONDITIONAL (гипотеза, требует проверки)
```
Гипотеза: φ(carry[i], carry[j] | W ∈ Sol_Wang) ≠ 0

То есть carry-корреляции существуют, но только в подпространстве Sol_Wang.
Если верно — carry-граф является СЛЕДСТВИЕМ Wang-структуры, не причиной.
Проверка: измерить φ на 2000 Wang-парах из предыдущих сессий.
```

### T_COMPOSITE_OBJ
```
Composite objective = hw59 + 25*(1-csync) + 10*(6-inv)/6
работает лучше чем hw59 alone:
  - Достигает inv=6/6 при hw59<100 (впервые)
  - N3 достигает hw59=90 с inv=6/6 за 621K итераций
  - Без composite: inv=6/6 при hw59=117..138 (Zone-2 сессия)
```

## 4. ДИАГНОЗ CARRY-РЕЗОНАНСА

**Почему сигналы из Zone-2 сессии оказались шумом при проверке?**

Zone-2 сессия измеряла корреляции на ПАРАХ (W, W') где W' = best Wang-SA результат. Это filtered subspace — не случайный W.

Аналогия с RIPEMD: нейтральные биты M[10..15] реальны, но только при конкретном дифференциале ΔM[0]. При случайном ΔM они нейтральны по другим причинам.

**Carry-сигналы Zone-2 = корреляции условные на Wang-дифференциал.** Это ценно, но их нельзя использовать как безусловный objective.

**Правильное применение (новая идея):**
```
Carry-conditional SA:
1. Сначала найти W с Wang-структурой (hw59<100)
2. ПОТОМ использовать carry-граф как refinement operator
3. carry-корреляции работают ВНУТРИ Wang-пространства

Это объясняет почему ACB (Серия VI) не работал как независимый инструмент:
он применялся до Wang-структуры, а не после.
```
