## ГЛАВНАЯ ТЕОРЕМА П-129

### T_WANG_IS_SUBTRACTION (доказана аналитически + верифицирована 1000/1000)

```
Sensitivity[r] = d(ef[r+1])/d(wf[r]) = 1  точно  для r=0..15

Доказательство:
  ef[r+1] = df[r] + T1f
  T1f = hf[r] + S1(ef[r]) + Ch(ef,ff,gf) + K[r] + W64f[r]
  W64f[r] = wf[r]  для r<16  (прямое, без расписания)
  d(T1f)/d(wf[r]) = 1  →  sensitivity = 1  QED

Следствие:
  wf[r] = wn[r] - de_before[r+1]  для r=1..15
  wf[0] = wn[0] + 0x8000

Wang-адаптация = простое вычитание de_before.
dW[r] = -de_before[r+1]  точно.
```


## ИТОГ — ЧТО ДОКАЗАНО

```
Фундаментальная структура Wang-пары:
  (Wn, Wf) — Wang-пара iff:
    Wf[0] = Wn[0] + 0x8000
    Wf[r] = Wn[r] - de_before[r+1]  r=1..15

Это ТОЧНАЯ алгебраическая характеристика
всего пространства Wang-пар.

Барьер r=17: de_before[17] = de[17] случаен.
Из T_WANG_IS_SUBTRACTION: нет свободного параметра
для управления de[17] после фиксации Wf[0..15].
T_DOF_SATURATION подтверждена через алгебру.

Единственный путь к полной коллизии SHA-256:
Найти структуру в de[17] через выбор Wn.
Но de[17] — это результат 17 нелинейных раундов
от случайного Wn → равномерно случаен.
```
