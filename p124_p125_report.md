## НОВЫЕ ТЕОРЕМЫ

**T_DOF_SATURATION** (П-125, аналитически):
```
Wang-цепочка использует РОВНО 16 степеней свободы dW[0..15]
для 16 уравнений de[1..16]=0.
Якобиан системы вырожден mod 2 (P=0/2000).
Барьер r=17 = граница линейной алгебры системы.
Нет остаточных степеней свободы для r>=17.
```

**T_XOR_W64_SPARSE** (П-125, аналитически):
```
Для XOR дифференциала dW[0]=2^b:
  dW64[17]_xor = 0  для ВСЕХ b=0..31
  dW64[18]_xor = s1(2^b)  HW=2..3  (очень разреженный!)

XOR расписание линейно: dW64[r]_xor = s1(dW64[r-2]) XOR ...
Нет carries -> полностью предсказуемо.
```

**T_SPARSE_W64_OPROVERGNUT** (П-125):
```
P(de[18]=0 | HW(dW64[18])=k) = 0 для всех k на 5M Wang-пар.
Биномиальная структура dW64[18] не влияет на P(de[18]=0).
```
