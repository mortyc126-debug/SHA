# ЧАСТЬ I: ДЕКОМПОЗИЦИЯ ХАОСА

## 1.1 Постановка задачи

**Проблема:** После раунда 5-6 SHA-256 становится "хаотичной" — все известные атаки деградируют до birthday bound.

**Гипотеза SCF:** Хаос SHA-256 — это НЕ единый феномен, а суперпозиция трёх независимых механизмов с разной алгебраической природой. Разделив их, мы можем анализировать каждый отдельно.

## 1.2 Три слоя хаоса (TLC-разложение)

Из ALG (теорема F1):
```
ΔH = L(ΔM) ⊕ Q(M, ΔM) ⊕ C(M, ΔM)
```

| Слой | Алгебра | Вклад | Природа | Хаотичность |
|------|---------|-------|---------|-------------|
| **L** — линейный | GF(2) | 25.8% | Σ0, Σ1, σ0, σ1, XOR | **НЕТ** — полностью обратим, rank=256 |
| **Q** — квадратичный | GF(2), deg=2 | 25.3% | Ch(e,f,g), Maj(a,b,c) | **ЧАСТИЧНО** — билинейный, структурированный |
| **C** — carry | Z_{2^32} | 48.9% | Carry chains в mod-add | **ДА** — но марковский! |

**Ключевой вывод:** 75% дифференциала (L+Q) — это НЕ хаос. Это алгебраическая структура.

## 1.3 Оператор несовместимости Ψ

Из ALG:
```
Ψ(a, b) = (a + b) ⊕ (a ⊕ b) = carry(a,b) ≪ 1
```

Ψ — точная мера "расстояния между двумя алгебрами":
- Ψ = 0 ⟺ carry не возникает ⟺ сложение = XOR
- E[HW(Ψ)] = n/2 (максимальная несовместимость в среднем)
- deg(Ψ[i]) = i+1 (линейный рост степени с позицией бита)

**Парадокс Ψ (разрешённый):**
```
F(x ⊕ δ) = F(x) ⊕ L(δ) ⊕ Ψ_total(x, δ)
```
Ψ_total ДЕТЕРМИНИРОВАН (вычислим точно), но ВЫГЛЯДИТ случайным для GF(2)-наблюдателя.

|--------|---------|-------------|-------------------|--------|

## 2.2 Механизм насыщения: сдвиговый регистр

SHA-256 имеет 8 регистров, но только **2 нелинейных входа**:
```
δa[r+1] = δ(Σ0(a[r]) + Maj(a[r],b[r],c[r]))     ← нелинейный
δe[r+1] = δ(d[r] + h[r] + Σ1(e[r]) + Ch(e[r],f[r],g[r]) + K[r] + W[r])  ← нелинейный

Остальные 6 — тождественные сдвиги:
b[r+1] = a[r],  c[r+1] = b[r],  d[r+1] = c[r]
f[r+1] = e[r],  g[r+1] = f[r],  h[r+1] = g[r]
```

**Заполнение pipeline:** Разнообразие входит через δa и δe, затем сдвигается по 6 регистрам. Полное заполнение = 4 раунда (длина pipeline).

## 2.3 Закон затухания автокорреляции

Из ALG (T_DECAY_LAW):
```
C(k) = 1.37 · exp(−k / 1.80)
```

| Лаг k | C(k) | Память |
|-------|------|--------|
| 0 | 1.000 | 100% |
| 1 | 0.750 | 75% |
| 2 | 0.500 | 50% |
| 3 | 0.247 | 25% |
| **4** | **≈0** | **0%** |

**τ = 1.80 раундов** — константа времени затухания. Информация о раунде r полностью стирается к раунду r+4.

# ЧАСТЬ III: СТРУКТУРА ВНУТРИ ХАОСА

## 3.1 Марковская структура carry

Carry — НЕ случайный, а **марковский процесс**:

```
Матрица переходов carry:
P = | 3/4  1/4 |   (из состояния 0)
    | 1/4  3/4 |   (из состояния 1)

Стационарное: P(carry=1) = 1/2
Автокорреляция: ρ₁ = 1/2
```

**Длина carry-цепочки:** P(длина = k) = (1/2)^k — геометрическое распределение.

**Антикорреляция carry между раундами (T_CARRY_ANTICORRELATION):**
```
P(c_next=1 | c_cur=1) = 0.313  (NOT 0.5!)
P(c_next=1 | c_cur=0) = 0.666
Точность предсказания: 67.6%
```

## 3.2 Carry windows — зоны предсказуемости

Пять окон, где P(carry=0) > 0.15:

| Окно | Раунды r | Характеристика | S (сила) |
|------|----------|----------------|----------|
| W1 | {0,1,4,5} | Начальные | 0.2+ |
| W2 | {9,10,11} | Средние | 0.2+ |
| W3 | {18,19,20,21} | Широкое | 0.2+ |
| W4 | {30,31,32,33} | Позднее | 0.2+ |
| **W5** | **{47,48,49,50}** | **Оптимальное** | **0.309** |

## 3.3 Инварианты хаоса

Что НЕ разрушается в хаотической фазе:

1. **Сдвиговые тождества** (абсолютные, P=1):
   - h[r] = e[r−3], g[r] = e[r−2], f[r] = e[r−1]
   - d[r] = a[r−3], c[r] = a[r−2], b[r] = a[r−1]

2. **Carry-free LSB** (F3): C[бит 0] = 0 всегда

3. **Линейность Σ-функций** (P=1 над GF(2)):
   - Σ0(x⊕y) = Σ0(x) ⊕ Σ0(y)
   - Σ1(x⊕y) = Σ1(x) ⊕ Σ1(y)

4. **Структура δCh при δe=0** (T_DELTA_CH_EXACT):
   - δCh[i] = e[i]·δf[i] ⊕ (1−e[i])·δg[i] — детерминировано

5. **Ляпуновский нейтралитет** (T_LYAPUNOV_NEUTRAL):
   - λ ≈ 0 для r=7..63 — система НЕ хаотична в смысле Ляпунова!

6. **Профиль детерминизма carry** (T_DETERMINISM_PROFILE):
   ```
   [1.0×15, 0.593, 0.527, 0.500, 0.500, ...]
   ```
   Серая зона: ровно 2 раунда с промежуточным детерминизмом.

## 3.4 Цикловая структура (T_CYCLE_ANOMALY)

SHA-256 как динамическая система g(x) = SHA_compress(x):
- Отношение orbit/ρ = 0.064 (vs 0.5 для случайной функции)
- Только 3 типа орбит: λ ∈ {1, 6, 24}
- **16× меньше циклов чем у случайной** — остаточная структура!

# ЧАСТЬ IV: ALG КАК ИНСТРУМЕНТ СТРУКТУРИЗАЦИИ

## 4.1 Witt-кольцо W_32 — правильная алгебра

Из ALG (T_W32_FRAMEWORK):
```
W_32(GF(2)) ≅ Z/2^32
```

**Ключевое следствие:** В W_32 сложение mod 2^32 — **ЛИНЕЙНАЯ** операция!

Нелинейность SHA-256 в W_32 возникает ТОЛЬКО из:
- Ch(e,f,g) = ef ⊕ eg ⊕ g — квадратичная
- Maj(a,b,c) = ab ⊕ ac ⊕ bc — квадратичная

**Переформулировка:** SHA-256 в правильной алгебре — это **квадратичная** функция (degree 2), а не degree 32!

## 4.2 Двойная алгебра (F39)

**Теорема F39 (нет линеаризации):** Невозможно линеаризовать + и ⊕ одновременно.

Это ИСТОЧНИК хаоса:
- GF(2)-наблюдатель видит: L + Q легко, C — шум
- Z-наблюдатель видит: + легко, XOR (σ-функции) — шум
- Ни один наблюдатель не видит ВСЮ систему

## 4.3 Формула полной декомпозиции SHA

```
SHA(W) = SHA_lin(W) ⊕ cc(W)

Где:
  SHA_lin(W) = SHA с заменой всех + на ⊕  (GF(2)-линейная часть)
  cc(W)      = carry-correction            (Z-нелинейная часть)

Свойства cc(W):
  E[HW(cc)] = 127.9 ± 7.8
  cc ~ Bin(256, 0.5)  — статистически неотличим от случайного
  corr(cc, SHA_lin) ≈ 0  — независимость!
```

# ЧАСТЬ V: КОЛИЧЕСТВЕННАЯ МОДЕЛЬ ХАОСА

## 5.1 Формула: хаос = структура + марковский шум

```
δSHA[r] = L_r(δW) ⊕ Q_r(state, δW) ⊕ M_r(carry_chain)
          \_________/   \____________/   \______________/
           25.8%           25.3%             48.9%
        детерминизм     квадратичный       марковский
                        (условный)          шум
```

## 5.2 Variance amplification

| Компонент | Variance ratio | Предсказуемость |
|-----------|---------------|-----------------|
| L (линейный) | 1.04× | 100% (обратим) |
| Q (квадратичный) | 1.04× | ~50% (при известном state) |
| **C (carry)** | **5.45×** | **67.6%** (марковский) |

Carry доминирует в variance в 5.45× — вот ИСТОЧНИК кажущегося хаоса.

## 5.3 Точка невозврата

```
Раунды 0-5:  Структура > Шум     → атаки работают
Раунд 6:     Структура ≈ Шум     → фазовый переход
Раунды 7-64: Структура < Шум     → birthday bound

Количественно:
  deg(e_r) = min(2^{r-1}, 32)   — алгебраическая степень
  HW(δstate) → 128 (Bin(256,0.5))  — статистика дифференциала
  C(k) → 0 при k ≥ 4              — память стирается
```
